# แผนก : ห้องอาหาร
## (ให้บริการอาหารตามความต้องการของลูกค้า)
### 1. ให้บริการด้านอาหาร
#### 1.1 รับออเดอร์จากลูกค้า
* พนักงานเสิร์ฟจดรายการอาหารลงในกระดาษ
* พนักงานเสิร์ฟแจ้งรายการอาหาร ให้กับแคชเชียร์
* แคชเชียร์คีย์รายการอาหารทั้งหมดเพื่อส่งต่อให้ครัวทำอาหาร
* ม้าเร็วนำอาหารมาส่งที่ห้องอาหาร
* พนักงานเสิร์ฟเช็คอาหาร ***หากไม่มีปัญหา*** ทำการจัดเสิร์ฟให้กับลูกค้า
***หากมีปัญหา (อาหารไม่ตรงตามที่สั่ง)*** เช็คสาเหตุว่าปัญหาเกิดที่ตรงไหน 

หากผิดที่ห้องครัว ห้องครัวต้องทำอาหารออกมาใหม่ และตัด shot ที่ห้องครัว

หากผิดที่แคชเชียร์คีย์อาหารไม่ตรงทำการตัด shot ที่แคชเชียร์ และคีย์อาหารเพื่อให้ห้องครัวทำออกมาใหม่ 

หากผิดที่พนักงานเสิร์ฟที่รับออเดอร์ไม่ตรงกับที่ลูกค้าสั่ง ตัด shot ที่พนักงานเสิร์ฟ และทำการจดบันทึกเพื่อส่งให้แคชเชียร์คีย์อาหารใหม่

#### 1.2 คูปองอาหารเช้า
* **ในกรณีทัวร์ หรือ กลุ่มคณะ** ไกด์จะได้รับคูปองอาหารเช้าตามจำนวนที่แจ้งเข้าพักกับทาง front 
เมื่อมีใช้บริการจะมีพนักงานเสริฟ? หัวหน้า? ยืนคู่กับไกด์และนับจำนวนคน ให้ตรงกับคูปองที่แจก
* **ในกรณีไม่ใช่ทัวร์**
ห้องอาหารจะได้รับใบแจ้งจาก front ซึ่งมีรายละเอียดของห้องที่ทำการซื้ออาหารเช้า 
เมื่อมีการใช้บริการพนักงานเสริฟ? ถามเลขห้องพักจากลูกค้า และเช็คตามใบที่ front แจ้งรายละเอียด

### 2. ทำตามใบ Function งาน
* จัดเตรียมห้องอาหาร เก้าอี้ โต๊ะ แก้วน้ำ ช้อนส้อม ให้พอดีกับความต้องการของใบ Function งาน

# แผนก : Store
## (ดูแลรักษาของส่วนหนึ่งของโรงแรม)
### 1. รับของจากร้านค้า  
#### 1.1 รับของจากร้านค้าเข้า store (store เป็นคนเขียนใบสั่งซื้อ)
* store เขียนใบสั่งซื้อเพื่อส่งให้ฝ่ายจัดซื้อดำเนินการ
* เมื่อมีร้านค้ามาส่งสินค้า จะต้องมี **ฝ่ายจัดซื้อ และ store** เข้ามาร่วมตรวจสอบในการรับสินค้า
* เมื่อตรวจสอบสินค้าเรียบร้อย store ต้องยิงบาร์โค้ดสินค้า เพื่อกรอกจำนวนสินค้าที่รับเข้ามา
#### 1.2 รับของจากร้านค้าเข้า outlet (outlet เป็นคนเขียนใบสั่งซื้อ)  
* แต่ละ outlet เขียนใบสั่งซื้อเพื่อส่งให้ฝ่ายจัดซื้อดำเนินการ
* เมื่อมีร้านค้ามาส่งสินค้า จะต้องมี **ฝ่ายจัดซื้อ, store และ outlet นั้นๆ** เข้ามาร่วมตรวจสอบในการรับสินค้า

### 2. นำของออกจาก store
#### 2.1 นำของออกจาก store ให้ outlet
* แต่ละ outlet เขียนใบเบิกที่ store
* store จ่ายของตามใบเบิก
* store คีย์จำนวนสินค้าที่จ่ายออกลงในโปรแกรมเพื่อให้โปรแกรมตัดสต็อคสินค้า เพื่อดูสินค้าคงเหลือ
* store เช็คจำนวนสินค้าให้ตรงตามยอดคงเหลือ

### 3. ตรวจเช็คสต็อคสินค้า
#### 3.1 ตรวจเช็คสต็อคสินค้าของ store 
* ทุกๆวัน store ทำการสุ่มเช็คสินค้า (ทั้งเครื่องดื่ม และ ของฟุ่มเฟือย) อย่างละ 3 รายการ
* ทุกๆเดือนมีการ **ตรวจสอบสินค้าจากฝ่ายบัญชี** ทุกรายการ
#### 3.1 ตรวจเช็คสต็อคสินค้าของบาร์น้ำ
* store เข้าไปตรวจสอบตาม outlet ต่างๆเพื่อเช็คจำนวนสินค้าในบาร์น้ำ ให้ตรงกับจำนวนคงเหลือในโปรแกรม
* store เตรียมจำนวนสินค้าตามยอดจำหน่าย เพื่อให้ตรงกับพาร์ที่กำหนดไว้
